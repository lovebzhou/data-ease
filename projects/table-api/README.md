## QuickStart

### Installation

```bash
$ pnpm install
```

### Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

### Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

### repl

```bash
# 启动repl模式
pnpm start:repl

# 进入repl模式后，执行debug()会打印所有的 module 及之下的 controllers 和 providers
>debug()
```
